Funcionalidades del Panel

Visualizar Usuarios

Ver todos los usuarios registrados con paginación y búsqueda por nombre o email.

Crear Usuarios

Agregar nuevos usuarios al sistema con todos sus datos.

Editar Usuarios

Modificar información de usuarios existentes, incluyendo contraseñas.

Eliminar Usuarios

Eliminar usuarios del sistema con confirmación de seguridad.

Estadísticas

Ver estadísticas en tiempo real: total de usuarios, registros del día y de la semana.

Cómo Usar el Panel

  1. Acceso: Ingresa uno de los códigos de técnico en la página de login
  2. Navegación: Usa los botones de la interfaz para realizar acciones
  3. Búsqueda: Utiliza el campo de búsqueda para encontrar usuarios específicos
  4. Paginación: Navega entre páginas si hay muchos usuarios
  5. Acciones: Usa los botones de editar y eliminar en cada fila de usuario

Endpoints de la API

El panel utiliza los siguientes endpoints (requieren código de técnico en headers):

GET /api/tech/usuarios - Listar usuarios
GET /api/tech/usuarios/:id - Obtener usuario específico
POST /api/tech/usuarios - Crear usuario
PUT /api/tech/usuarios/:id - Actualizar usuario
DELETE /api/tech/usuarios/:id - Eliminar usuario
GET /api/tech/stats - Obtener estadísticas

Seguridad

Importante:
  • Todos los endpoints requieren un código de técnico válido
  • Las contraseñas se encriptan automáticamente
  • Las eliminaciones son permanentes
  • Todas las acciones se registran en los logs del servidor
